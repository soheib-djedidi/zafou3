Introduction à CSS

CSS (Cascading Style Sheets) définit l'apparence visuelle des pages web. Il a été normalisé par le W3C en 1996 et s'est enrichi de nombreuses fonctionnalités : animations, grilles, variables personnalisées ou encore mises en page responsive.

Structurer une feuille de style

Une règle CSS se compose d'un sélecteur et d'un bloc de déclarations : chaque déclaration associe une propriété à une valeur. 

            
article {
  background-color: #ffffff;
  border-radius: 16px;
  padding: 32px;
}

Lier une feuille CSS à une page HTML

On relie généralement une feuille de style externe avec la balise <link rel="stylesheet" href="chemin.css" /> placée dans l'en-tête du document. Cette méthode sépare le contenu (HTML) de la présentation (CSS) et facilite la maintenance en centralisant les styles au même endroit.

En utilisant un fichier CSS unique pour plusieurs pages, on garantit également une charte graphique cohérente et on réduit la quantité de code chargé par le navigateur grâce à la mise en cache.

Bonnes pratiques

  • Utiliser des classes explicites pour cibler les éléments.
  • Centraliser les couleurs, polices et espacements dans des variables ou des feuilles communes.
  • Travailler la responsivité avec des media queries pour s'adapter à chaque support.
  • Tester l'accessibilité : contraste des couleurs, taille des polices, interactions au clavier.
